# Mobile Screen

Use this skill for phone-sized UI prototypes.

## Rules

- Design for thumb reach, readable 16px base text, and 44px minimum tap targets.
- Include status, navigation, primary action, loading, and error states when the flow expects them.
- Do not cram desktop tables into mobile cards; reshape the workflow.
- Check text wrapping, safe areas, bottom controls, and modal height on small screens.
