# Live Site Visual Audit

Use this skill when the source of truth is a live website or screenshot.

## Rules

- Compare layout, density, typography, spacing, color, and interaction affordances.
- Do not copy branding or copyrighted assets unless the user owns them or explicitly provides them for the project.
- Preserve the target product workflow over superficial visual imitation.
- List differences by severity and implement the highest-impact fixes first.
