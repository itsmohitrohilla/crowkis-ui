# Frontend Debugging

Use this skill for targeted frontend bug hunts.

## Loop

1. Reproduce the problem in the browser or test runner.
2. Capture the error, route, viewport, and interaction path.
3. Inspect the smallest likely component or state boundary.
4. Patch the cause, not just the symptom.
5. Re-run the original reproduction and one nearby regression check.
