# Web Prototype

Use this skill for fast, realistic web prototypes.

## Rules

- Build the actual first screen, not a placeholder landing page about the app.
- Keep copy specific to the brief; avoid lorem ipsum and fake metrics.
- Use a clear responsive layout with no horizontal scroll or clipped text.
- Prefer a small number of strong sections over a long generic page.
- Use real images, generated images, or well-framed placeholders only when visual inspection is not required.
