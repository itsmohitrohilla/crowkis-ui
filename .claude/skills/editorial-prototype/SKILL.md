# Editorial Prototype

Use this skill for calm, text-forward interfaces.

## Rules

- Use generous whitespace, strong typographic contrast, and restrained color.
- Do not turn every section into a floating card.
- Prefer one memorable visual or typographic move over many decorative flourishes.
- Keep the page useful: navigation, hierarchy, and calls to action still matter.
